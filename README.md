# 🌐 Networking Lab Projects

This repository contains various networking configuration projects and practical labs. The purpose of this repository is to demonstrate hands-on networking skills, configuration knowledge, and troubleshooting experience.

## 📚 Projects Included

### 1. DHCP Configuration
- Configure DHCP server
- Create DHCP pools
- Assign IP addresses automatically to clients
- Verify DHCP lease allocation

### 2. DNS Configuration
- Configure DNS server
- Create forward lookup zones
- Configure hostname resolution
- Test DNS functionality using ping and nslookup

### 3. VLAN Configuration
- Create VLANs
- Assign switch ports to VLANs
- Configure trunk ports
- Verify VLAN communication

### 4. Inter-VLAN Routing
- Configure Router-on-a-Stick
- Enable communication between VLANs
- Verify routing functionality

### 5. Static Routing
- Configure static routes
- Verify network reachability
- Troubleshoot routing issues

### 6. OSPF Routing
- Configure OSPF routing protocol
- Advertise networks
- Verify neighbor relationships
- Check routing tables

### 7. NAT Configuration
- Configure Static NAT
- Configure Dynamic NAT
- Configure PAT (Port Address Translation)
- Verify internet connectivity

### 8. Access Control Lists (ACL)
- Standard ACL Configuration
- Extended ACL Configuration
- Traffic filtering and security implementation

### 9. SSH Configuration
- Secure remote device access
- Configure local users
- Generate RSA keys
- Verify SSH connectivity

### 10. Network Troubleshooting
- Ping
- Traceroute
- IP Configuration Verification
- Routing Table Analysis
